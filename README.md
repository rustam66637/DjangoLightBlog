# DjangoLightBlog

## Блог на Django, + Bootstrap.

Функциональность: <br>
  создание; <br>
  изменение; <br>
  удаление **постов** и **тегов**. <br>

Также: админская панель на ***CSS***, ограничение доступа незарегистрированных пользователей, <br>
постраничная навигация, поиск по постам и тегам. :shipit:
